Student Age Analyzer
Seatwork | April 15, 2026 | BSIT 2-3
A Java program that takes a student's name and age as input, then categorizes them using a separate static method getAgeCategory() — demonstrating the use of methods in OOP.
  • Child: age 0–12
  • Teenager: age 13–19
  • Adult: age 20–59
  • Senior Citizen: age 60+
  • Invalid Age: negative or out-of-range values
public static String getAgeCategory(int age) {
  if (age >= 0 && age <= 12) { return "Child"; }
  else if (age >= 13 && age <= 19) { return "Teenager"; }
  else if (age >= 20 && age <= 59) { return "Adult"; }
  else if (age >= 60) { return "Senior Citizen"; }
  else { return "Invalid Age"; }
}

  • Reference variables (acc1, acc2, acc3) are stored in the Stack

Activity 3 — Basic ATM System
Java Program | BSIT 2-3 | Eclipse IDE
A menu-driven ATM simulation using a do-while loop and switch-case. Users can check balance, deposit, and withdraw — with input validation and a ₱2,000 withdrawal limit per transaction. 
double balance = 5000;
do {
  // Display menu options 1–4
  switch (choice) {
    case 1: // Check Balance
    case 2: balance += depositAmount; // Deposit
    case 3: // Withdraw — limit: ₱2,000 max
      if (withdrawalAmount > 2000)
        System.out.println("Withdrawal limit exceeded!");
      else balance -= withdrawalAmount;
    case 4: // Exit
  }
} while (choice != 4);
  • do-while ensures the menu displays at least once before checking exit condition
  • switch-case cleanly separates the four transaction types
  • Input validation prevents invalid deposits and over-limit withdrawals
"This activity helped me understand how loops maintain program flow and how switch-case makes code cleaner and easier to manage for menu-based programs."

Activity 5 — Personal Expense Tracker
Java Program | April 16, 2026 | BSIT 2-3
A modular Java application that uses static methods to organize code into logical units: displayTitle(), calculateTotalExpenses(), checkBudgetStatus(), and displaySummary(). Demonstrates the use of methods — a core OOP principle. 
public static double calculateTotalExpenses(
    double food, double transport, double other,
    double utilities, double entertainment) {
  return food + transport + other + entertainment + utilities;
}

public static String checkBudgetStatus(double total, double budget) {
  if (total > budget)
    return "⚠ WARNING: Budget Exceeded by PHP " + (total - budget);
  else
    return "✔ GOOD JOB: You are within your budget!";
}
  • Static methods allow code reuse without creating objects
  • Separating logic into methods makes the code modular and readable
  • printf() with format specifiers ensures clean, aligned output
  • Demonstrated budget analysis with conditional feedback messages
